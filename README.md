# SmartCart

A smart shopping cart application built with Jetpack Compose.

This application provides a seamless and modern shopping experience, allowing users to sign in, scan items, and manage their shopping cart with ease. The UI is built entirely with Jetpack Compose, and the application is structured with a single-activity architecture, using Jetpack Navigation for screen transitions.

## Features

*   **Splash Screen:** A visually appealing entry point to the application.
*   **Authentication:** Sign-in and sign-up screens for user authentication.
*   **QR Code Scanning:** A dedicated screen for scanning QR codes on shopping carts.
*   **Home Screen:** A central hub for users to browse deals and promotions.
*   **Shopping Cart:** A screen to view and manage items in the cart.
*   **Checkout:** A streamlined checkout process with a payment summary.
*   **Profile:** A user profile screen with options for editing and viewing order history.

## Technologies Used

*   [Kotlin](https://kotlinlang.org/)
*   [Jetpack Compose](https://developer.android.com/jetpack/compose)
*   [Jetpack Navigation](https://developer.android.com/guide/navigation)
*   [Material Design 3](https://m3.material.io/)
